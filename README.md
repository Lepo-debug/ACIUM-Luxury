# ACIUM-Luxury
Um Site que simula uma loja online de joias com dashboard, pesquisa, filtragem, traduçao de site, tema claro/escuro, painel de administrador, login e cadastro, criaçao, ediçao e eliminaçao de itens e funcionarios, historico e criaçao de documento execel para melhor analize de dados, com ligaçao ao banco de dados Firebase
